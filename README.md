# BedrockLib

## Briefly about library

##### This library is ultimate utility pack for work with Spigot API. It is included Command API for flexibility working with commands in plugin, UI utilities for smart controlling menus and own economy system.

## Installation

##### For Maven:

```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```

```xml
<dependency>
  <groupId>com.github.MrBedrockpy</groupId>
  <artifactId>BedrockLib</artifactId>
  <version>1.0</version>
</dependency>
```

##### For Gradle:

```gradle
repositories {
  mavenCentral()
  maven {
    url 'https://jitpack.io'
  }
}
```

```gradle
dependencies {
  implementation 'com.github.MrBedrockpy:BedrockLib:Tag'
}
```
