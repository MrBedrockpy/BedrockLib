## BedrockLib
### Installation

For Maven:

```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```

```xml
<dependency>
  <groupId>com.github.MrBedrockpy</groupId>
  <artifactId>BedrockLib</artifactId>
  <version>1.0</version>
</dependency>
```

For Gradle:

```gradle
repositories {
  mavenCentral()
  maven {
    url 'https://jitpack.io'
  }
}
```

```gradle
dependencies {
  implementation 'com.github.MrBedrockpy:BedrockLib:Tag'
}
```
