# BedrockLib

BedrockLib is my library for work with Spigot API and more.

## Installation

To import this library into your project, you need to go to the configuration file of your Maven assembly and put this inside the "repositories" tag:

``` 
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>
```

And then inside the "dependencies" tag it is:

```
<dependency>
  <groupId>com.github.User</groupId>
  <artifactId>Repo</artifactId>
  <version>Tag</version>
</dependency>
```
